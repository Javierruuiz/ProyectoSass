    General

    css

    .container

    .container { ... }

    Description

    .container

    Establece un contenedor de ancho máximo de 1200px, centrado en la página, con un pequeño relleno interno de 1rem.

    Utilizado comúnmente para envolver el contenido y darle un ancho adecuado.

    .section

    .section { ... }

    Description

    .section

    Aplica un relleno vertical de 4rem en la parte superior e inferior del elemento. También extiende el mixin o placeholder %text-center, que se asume que alinea el contenido de forma central.

    Este estilo se utiliza para crear secciones destacadas con un espaciado adecuado y contenido centrado.

    img

    img { ... }

    Description

    img

    Establece que las imágenes no superen el ancho de su contenedor, manteniendo la proporción de la imagen con el ajuste automático de la altura. También se asegura de que las imágenes se muestren como bloques, eliminando cualquier espacio por debajo de la imagen, y aplica un borde redondeado de 8px.

    *

    * { ... }

    Description

    • Establece un estilo global para todos los elementos del documento. Elimina el margen y el relleno por defecto de todos los elementos, y asegura que el box-sizing se configure como border-box, lo que significa que el ancho y el alto de los elementos incluirán el relleno y los bordes.

    body

    body { ... }

    Description

    body

    Establece la fuente principal del cuerpo del documento mediante la variable $font-primary, el color del texto mediante la variable $text-color, y el color de fondo utilizando la variable $background-color. También se establece una altura de línea de 1.6 para mejorar la legibilidad.

    h1, h2, h3

    h1, h2, h3 { ... }

    Description

    h1, h2, h3

    Establece que todos los encabezados de nivel 1, 2 y 3 (h1, h2, h3) utilicen la fuente secundaria definida por la variable $font-secondary. También se les aplica un margen inferior de 1rem para separar adecuadamente los encabezados del contenido que sigue.

    h1

    h1 { ... }

    Description

    h1

    Establece el tamaño de fuente para el encabezado h1 utilizando una función font-size(xxl), que probablemente esté definida para proporcionar un tamaño de fuente grande y destacado.

    h2

    h2 { ... }

    Description

    h2

    Establece el tamaño de fuente para el encabezado h2 utilizando una función font-size(xl), que ajusta el tamaño de la fuente a un tamaño menor que el de h1.

    p

    p { ... }

    Description

    p

    Establece el tamaño de fuente para los párrafos con la función font-size(base), que ajusta el tamaño de la fuente al valor base para el cuerpo de texto. Además, se configura una altura de línea de 1.6 para asegurar una lectura cómoda y clara.

    Description

    Colores principales

    • $primary-color: Color principal, utilizado para elementos destacados o botones.

    Fuentes

    • $font-primary: Fuente principal utilizada para los textos principales.
    • $font-secondary: Fuente secundaria utilizada para textos decorativos o secundarios.

    font-sizes

    $font-sizes: (
  small: 0.875rem,
  base: 1rem,
  large: 1.25rem,
  xl: 1.5rem,
  xxl: 2rem
);

    Description

    Tamaños de fuente

    • $font-sizes: Mapa que contiene los tamaños de fuente.
      • small: 0.875rem; Tamaño de fuente pequeño.
      • base: 1rem; Tamaño de fuente base (por defecto).
      • large: 1.25rem; Tamaño de fuente grande.
      • xl: 1.5rem; Tamaño de fuente extra grande.
      • xxl: 2rem; Tamaño de fuente muy grande.

    Used by

    • [function] font-size

    breakpoints

    $breakpoints: (
  mobile: 480px,
  tablet: 768px,
  desktop: 1024px
);

    Description

    Breakpoints

    • $breakpoints: Mapa que contiene los valores de los puntos de ruptura (breakpoints) para la respuesta.
      • mobile: 480px; Punto de ruptura para dispositivos móviles.
      • tablet: 768px; Punto de ruptura para tablets.
      • desktop: 1024px; Punto de ruptura para escritorios.

    Used by

    • [function] breakpoint
